EasyTrip — Documentazione prodotto (codice)

Questa cartella contiene documenti HTML di visione e specifiche. La versione attuale del prodotto è implementata nel repository easytrip-saas con lo stack sotto.

Stack attuale (no no-code)

  • Next.js — UI e API Routes
  • Prisma + PostgreSQL — dati (Trip, TripVersion, Day, Payment, …)
  • Clerk — autenticazione
  • Stripe — acquisto viaggio e rigenerazioni a pagamento (€1,99)
  • Inngest — job asincrono generazione itinerario
  • Anthropic (Claude) — generazione JSON strutturato + sostituzione slot
  • Resend (opzionale) — email transazionali via API REST
